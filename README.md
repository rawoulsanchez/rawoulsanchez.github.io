# RawoulSanchez.github.io

This repo hosts the code for the portfolio site.

## Structure

- `index.html` — main web page
- `images/` — all poster assets used on the film shelves
- `ASSET-MANIFEST.md` — lists all required poster asset filenames

## Notes

All posters follow a strict naming convention:

### Vertical posters (9:16)
poster-vert-A1.png … poster-vert-C10.png

### Horizontal posters (16:9)
poster-wide-A1.png … poster-wide-C10.png

Only some assets currently exist. Missing ones will be added later.
